\chapter{Algebra and Functions: Part I}

Functions are a big part of the SAT math section. In general, we use the symbols $f(x)$ (or $y$) to indicate a function of the input $x$. A \textit{function} is defined as a set of rules which maps to each input one and only one output. Figure \ref{fig:function_figure} below shows a mapping between two sets where each input has exactly one output. Figure \ref{fig:relation_figure} is an example of a mapping between two sets that are not functions because $f(a)$ has two outputs: 2 and 4.
